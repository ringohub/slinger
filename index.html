Taut

What?

User scripts and stylesheets in your Slack.app for OS X.

Why?

Because I got really annoyed that ⌘ + W closed the entire Slack window instead of the current channel/group/message/whatever.

How?

Short and sweet: 

    $ wget https://zachsnow.github.io/taut/patch.sh && bash patch.sh

This patches your copy of Slack.app to load taut.js at load time. This script subsequently loads taut.css.

If you'd prefer, you can patch Slack to load whatever user script you'd like. Simply pass the URL of the script as an argument: 

    $ bash patch.sh https://someotherdomain.com/some-other-script.js

Caveats: the script must be served over HTTPS, with a CA-signed certificate (otherwise the underlying WebView will not deign to load it). Furthermore the entire URL must be less than 92 characters long, because I'm not actually a reverse engineer.
